One-click cache control

Manually flush the cache when needed

Marketing releases and emergency fixes can invalidate cache globally with a single action—whether you run on the Content.One CDN powered by Fastly or pair us with Akamai, CloudFront, or Cloudflare.

# Purge a campaign path across Content.One + partner CDNs
curl -X POST https://api.contentone.com/v1/cache/purge \
  -H "Authorization: Bearer <token>" \
  -d '{ "paths": ["/campaigns/spring/*"], "cascade": ["akamai", "cloudfront"] }'
