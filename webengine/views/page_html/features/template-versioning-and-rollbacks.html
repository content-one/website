Command palette

Manage versions in code or from the browser.

Use Content.One’s command palette or CLI to snapshot templates, promote changes between environments, or roll back instantly.

  • Run `zesty pull` & `zesty push` to sync templates locally.
  • Snapshot environments before publishing major changes.
  • Trigger rollbacks via API for automated recovery plans.
Automation snippets 
# Pull environment to local workspace
zesty pull --instance 8-abc-123 --models --views

# Snapshot templates before editing
zesty snapshot create --label "pre-campaign-launch"

# Push updates back to Content.One
zesty push --views --assets

POST https://accounts.api.zesty.io/v1/instances/:instance_zuid/rollback
{
  "view_zuid": "8-abc-123",
  "target_version": "2024-02-10T18:42:00Z",
  "notes": "Rollback landing page before campaign"
}
git checkout -b feature/header-update
# Edit locally in VS Code, run tests
git commit -am "Update hero template"
git push origin feature/header-update
# Open PR -> merge -> Content.One deploys via webhook
